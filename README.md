# Conway's Game of Life


## Folders
```
root          Root directory
├─ gol-wasm   Game of Life WASM and JavaScript API
└─ www        Web application
```

## tl;dr
Install [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/), then;
```shell
cd wasm/
wasm-pack build
cd ../www/
npm i
npm run serve
```
